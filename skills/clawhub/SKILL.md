Use this skill when you need to interact with the ClawHub skill registry.

## When to Use

- Search for available skills by keyword or category
- Get detailed information about a specific skill
- List all installed or available skills
- Check skill compatibility or requirements

## Tool Flow

This skill uses the ClawHub MCP tools to:
1. Search the registry for skills matching criteria
2. Retrieve skill metadata and documentation
3. Provide usage guidance for found skills

## Output

Returns structured information about matching skills including:
- Skill name and description
- Version and compatibility info
- Installation status
- Usage instructions
