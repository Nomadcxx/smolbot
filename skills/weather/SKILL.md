Use this skill when you need weather information.

## When to Use

- Check current weather conditions for a location
- Get weather forecast for planning purposes
- Get weather context for travel or events
- Check severe weather alerts

## Input

Provide a location (city name, airport code, or coordinates).

## Output

Returns weather information including:
- **Current conditions**: Temperature, feels-like, humidity, wind
- **Forecast**: Daily high/low, precipitation chance, conditions
- **Alerts**: Any active weather warnings

## Weather Codes

Common conditions represented:
- Clear/Sunny
- Partly cloudy
- Cloudy
- Rain
- Heavy rain
- Snow
- Thunderstorm
- Fog
- Wind

## Tips

- Specify timezone if relevant for forecast timing
- Use airport codes for precise location (e.g., JFK, LAX)
- Check extended forecast for planning events
